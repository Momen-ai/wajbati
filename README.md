# مشروع وجباتي (Wajbati) - منصة الأسر المنتجة 🍽️

مشروع تخرج متكامل لربط الأسر المنتجة (الخبايا) والزبائن لطلب الطعام المنزلي الطازج.

## 🚀 المميزات الأساسية
- **نظام أدوار متكامل:** (مسؤول، طباخ، زبون).
- **إدارة الوجبات:** إضافة، تعديل، وحذف الوجبات مع الصور والتصنيفات.
- **دورة حياة الطلب:** تتبع الطلب من (قيد الانتظار) إلى (تم التوصيل).
- **نظام الإشعارات:** إشعارات فورية للطباخ عند استلام طلب جديد، وللزبون عند تغيير حالة الطلب.
- **نظام التقييم والمفضلة:** إمكانية تقييم الوجبات وإضافتها للمفضلة.
- **تعدد اللغات:** دعم كامل للغتين العربية والإنجليزية.

## 🛠️ المتطلبات
- PHP >= 8.2
- Composer
- MySQL
- Node.js & NPM

## ⚙️ خطوات التشغيل
1. قم بفك الضغط عن المشروع.
2. افتح الوحدة الطرفية (Terminal) في مجلد المشروع وقم بتثبيت الحزم:
   ```bash
   composer install
   npm install
   npm run dev
   ```
3. قم بإنشاء ملف `.env` وقم بضبط إعدادات قاعدة البيانات الخاصة بك.
4. قم بإنشاء قاعدة بيانات جديدة ثم قم بتهجير الجداول وزرع البيانات:
   ```bash
   php artisan migrate --seed
   ```
5. قم بتشغيل خادم التطوير:
   ```bash
   php artisan serve
   ```

## 📝 ملاحظات تقنية
- المشروع يستخدم **Laravel 11**.
